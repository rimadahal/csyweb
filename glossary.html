Web development terms:
1.HTML : https://www.webopedia.com/TERM/H/HTML.html The full form of HTML is Hyper Text Markup Language and it is used on world wide web to make documents. To make a web page, the structure starts and ends with html tag. All the documents are created inside the body tag.
2.CSS : https://techterms.com/definition/css The full form of css is cascading style sheets. For making a weg page, css is used for the layout formation. To create different pages for a website, cascadind style sheets are used for making the pages uniform. CSS can be used to manage various properties like; font size, background colour, height, width, position, etc. that's why webpages integrate with css.
3.Flex property : https://www.geeksforgeeks.org/css-flex-property/ In css, flex properties are used to manage the length of different flexible items, and it is also used to make the web page responsive. The different types of flex properties in css are flex-grow, flex-shrink and flex-basis.
4.Java Script : https://techterms.com/definition/javascript Java script is a programming language used to add interactive elements to the webpage. The functions of java scripts are added within the script tag.
